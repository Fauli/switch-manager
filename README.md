## Introduction

Switch Manager is a simple tool to manage your network switches.

## Features

- Nicely show all your switches
- SSH to your switches
- Ping your switches
- Trace route to your switches

## Installation

```bash
source venv/bin/activate
pip install -r requirements.txt 
```
